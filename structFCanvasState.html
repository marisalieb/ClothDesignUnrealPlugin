Detailed Description

Represents the complete state of the cloth design canvas at a given moment.

This struct encapsulates all relevant information required to render the canvas, manage user interactions, undo/redo operations, and handle sewing operations. The intent is to provide a single, serialisable snapshot of the canvas state so that operations such as undo, redo, and temporary previews can be performed consistently without relying on scattered state.
