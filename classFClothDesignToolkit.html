Detailed Description

Toolkit that provides the Cloth Design editor's UI and tool palettes.

The toolkit bridges the editor mode and Slate UI: it initialises in-editor panels, exposes inline content for embedding in the host editor, and centralises command wiring and object picker state. Keeping toolkit logic here keeps the mode class lightweight and makes UI tests easier to write.
