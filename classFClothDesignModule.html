Detailed Description

Main plugin module class for the Cloth Design editor mode.

This module handles initialization and shutdown of the Cloth Design plugin, as well as spawning and managing custom Slate UI for cloth pattern design.

Responsibilities:

  • Registers editor tabs and menus.
  • Spawns the 2D design window and manages its lifecycle.
  • Handles user actions such as generating meshes and sewing patterns.
