Detailed Description

Represents a single cloth pattern mesh actor in the scene.

This actor encapsulates a procedural mesh component along with metadata for sewing, boundary sampling, and vertex alignment. It maintains both 2D pattern information and corresponding 3D mesh data, allowing seamless interaction between the design canvas and the runtime mesh.
