Detailed Description

Handles triangulation and procedural mesh generation from canvas shapes.

Note
Thesis reference: See Chapter 4.6 for detailed explanations.

This class centralises all operations needed to convert user-drawn 2D shapes into fully triangulated 3D meshes. By encapsulating this functionality, it ensures that mesh construction is consistent, reusable, and maintainable.
